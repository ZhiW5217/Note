# django-项目结构
```
py2006_pro           ---------- 项目名

	py2006_pro       ---------- Django核心包
		__init__.py  ---------- 标记所在的目录是python的包
		asgi.py      ---------- django3.0之后支持的项目部署方式
		wsgi.py      ---------- django3.0之前唯一的一种项目部署方式
		
		settings.py  ---------- django核心配置文件
		urls.py      ---------- django的项目路由管理器
	
	templates        ---------- 用来存放 .html 网页
	
	manage.py        ---------- django管理器

```
